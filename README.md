# Security Engineer Plugin for Claude Code

Application security -- STRIDE threat modeling, OWASP audit, pen test planning, and compliance validation.

## Installation

```
/plugin marketplace add nagisanzenin/claude-code-plugins
/plugin install security-engineer@nagisanzenin
```

## Features

- STRIDE threat modeling
- OWASP Top 10 security audit
- Penetration test planning
- Compliance validation (SOC2, HIPAA, GDPR)
- Security scanning integration
